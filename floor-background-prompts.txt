GAUNTLET TOWER — BATTLE BACKGROUND IMAGE PROMPTS
Post-game floors 51-500  (one prompt per 10-floor sub-tier — 45 prompts total)
================================================================================

HOW TO USE
  - Each numbered block below is a COMPLETE, standalone prompt.
  - Just copy one whole block (the text after the "PROMPT:" line) and paste it
    into your image generator. No need to combine anything.
  - Generate at 16:9 widescreen (e.g. 1920x1080 / 1344x768).
  - Save each result to /public using the matching filename shown, e.g.
    "floor 51-60.png", "floor 61-70.png", ... so it lines up with the existing
    "floor 1-10.png" ... "floor 50.png" assets.

ART-STYLE NOTES (already baked into every prompt below — here for reference)
  - Dark high-fantasy painted game art, matching the existing floor backdrops.
  - It is a BATTLE ARENA BACKDROP: wide, symmetrical, with an empty open floor
    in the foreground (characters get drawn on top of it later).
  - No characters, no creatures, no text, no UI — environment only.
  - Moody and dim; the game lays a dark gradient over the top, so keep the
    centre from being too bright or busy.
  - Atmospheric depth: a glowing focal point or doorway in the mid-background.
